# Tuples

## What is a tuple?
A tuple is an ordered, immutable sequence.  
Use tuples when data should not be modified after creation.

```python
t = (10, 20, 30)
```

## Access and slicing

```python
print(t[0])
print(t[-1])
print(t[0:2])
```

## Why tuples are useful

- Prevent accidental modification
- Can be used as dictionary keys (if elements are hashable)
- Good for fixed-size records like coordinates

## Packing and unpacking

```python
point = (2, 5)   # packing
x, y = point     # unpacking
```

## Single element tuple
Comma is mandatory.

```python
a = (5,)   # tuple
b = (5)    # int
```

## Tuple methods

```python
t = (1, 2, 2, 3)
print(t.count(2))
print(t.index(3))
```

## Interview pitfalls

- Forgetting comma in one-element tuple
- Assuming full deep immutability (nested mutable items can still change)

```python
t = ([1, 2], 3)
t[0].append(4)
print(t)  # ([1, 2, 4], 3)
```
