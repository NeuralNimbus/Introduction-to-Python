# Dictionary

## What is a dictionary?
A dictionary stores data as `key: value` pairs.  
It is one of the most important tools in Python interviews.

```python
student = {"name": "Asha", "score": 92}
```

## Key concepts

- Keys must be unique and hashable
- Values can be of any type
- Dictionary is mutable

## Accessing values

```python
print(student["name"])        # direct access
print(student.get("grade"))   # safe access, returns None if absent
```

## Add/update/remove

```python
student["grade"] = "A"       # add
student["score"] = 95         # update
student.pop("grade")          # remove
```

## Iteration

```python
for key, value in student.items():
    print(key, value)
```

## Dictionary comprehension

```python
squares = {x: x * x for x in range(5)}
```

## Interview pattern: frequency map

```python
freq = {}
for ch in "banana":
    freq[ch] = freq.get(ch, 0) + 1
```

## Complexity note
Average-case O(1) for insert, lookup, delete by key.

## Interview pitfalls

- `d[key]` on missing key raises `KeyError`
- Using mutable objects as keys
- Forgetting to use `get()` for safe counting logic
