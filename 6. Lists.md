# Lists

## What is a list?
A list is an ordered, mutable collection.  
It can store values of different types and allows duplicate elements.

```python
a = [10, 20, 30, 40]
```

## Accessing elements
Lists support positive and negative indexing.

```python
print(a[0])   # 10
print(a[-1])  # 40
```

## Updating elements (mutability)

```python
a[1] = 25
print(a)  # [10, 25, 30, 40]
```

## Important methods

- `append(x)`: add at end
- `insert(i, x)`: add at index
- `pop([i])`: remove and return element
- `remove(x)`: remove first occurrence of value
- `sort()`: sort in place

```python
a.append(50)
a.insert(1, 15)
a.pop()
a.remove(30)
```

## Slicing
Slicing creates a new list.

```python
b = a[1:4]
rev = a[::-1]
```

## Copy vs alias

```python
a = [1, 2, 3]
b = a         # same object
c = a.copy()  # new object
```

## List comprehension
A compact way to build lists.

```python
squares = [x * x for x in range(6)]
evens = [x for x in range(10) if x % 2 == 0]
```

## Complexity notes

- Index access: O(1)
- Append: amortized O(1)
- Search: O(n)
- Insert/delete middle: O(n)

## Interview pitfalls

- Using `sort()` and expecting a new list (it returns `None`)
- Calling `index()` for missing values
- Creating matrix as `[[0] * m] * n` (shared rows)
