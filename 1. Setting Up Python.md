# Setting Up Python (Windows + VS Code)

## Why this matters
A broken setup wastes interview practice time. This chapter keeps setup minimal and reliable.

## 1. Install Python

1. Download Python from [python.org/downloads](https://www.python.org/downloads/).
2. During install, check `Add Python to PATH`.
3. Choose `Install Now`.

## 2. Verify installation

Open PowerShell and run:

```bash
python --version
pip --version
```

If both commands print versions, setup is correct.

## 3. Install VS Code + Python extension

1. Install VS Code from [code.visualstudio.com](https://code.visualstudio.com/).
2. Open Extensions.
3. Install `Python` (publisher: Microsoft).

## 4. Run your first file

Create `test.py`:

```python
print("Hello, Python")
```

Run:

```bash
python test.py
```

Expected output:

```text
Hello, Python
```

## Interview best practices

- Prefer running with terminal command `python file.py` over code-runner shortcuts.
- Keep one Python version for practice to avoid confusion.
- Learn to use `python -m pip install <package>`.

## Common pitfalls

- `python` command not found: PATH is not configured.
- Multiple Python installs causing package mismatch.
- Running code in a different interpreter than VS Code selected interpreter.
