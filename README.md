# Python for SDE-1 Interviews

This repository is beginner-first Python study material designed for SDE-1 interview preparation.  
Each chapter explains the concept in simple language, then gives short examples and common interview pitfalls.

## How to Study

1. Read one chapter at a time in order.
2. Understand the concept section first.
3. Run the examples and modify them.
4. Write 2-3 small programs from memory.
5. Revise interview pitfalls before mock tests.
