# Strings

## What is a string?
A string is an immutable sequence of characters.

```python
s = "hello"
```

Because strings are immutable, operations create new strings instead of modifying in place.

## Indexing and slicing

```python
s = "interview"
print(s[0])    # i
print(s[-1])   # w
print(s[:5])   # inter
print(s[::-1]) # weivretni
```

## Common string methods

- `strip()`: remove spaces/chars from both ends
- `split()`: split into list
- `join()`: join list into string
- `replace()`: replace substring
- `lower()`, `upper()`: case conversion

```python
text = "  python interview  "
print(text.strip())
print("a,b,c".split(","))
print("-".join(["a", "b", "c"]))
```

## Searching in strings

```python
s = "banana"
print("na" in s)      # True
print(s.find("na"))   # 2
print(s.count("na"))  # 2
```

## Formatting strings

```python
name = "Sam"
rank = 4
print(f"{name} rank: {rank}")
```

## Interview pitfalls

- Repeated `+` concatenation in loops (can be slow)
- Confusing `find()` and `index()` behavior
- Trying to modify a string by index
