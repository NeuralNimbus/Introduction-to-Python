# Sets

## What is a set?
A set is an unordered collection of unique elements.  
Sets are very useful when you need fast lookup and duplicate removal.

```python
s = {1, 2, 3}
```

## Key properties

- No duplicate elements
- Unordered (do not rely on index positions)
- Mutable container
- Elements must be hashable (e.g., no list inside set)

## Common operations

```python
s.add(4)
s.remove(2)      # raises KeyError if 2 is absent
s.discard(10)    # safe even if missing
x = s.pop()      # removes an arbitrary element
```

## Membership and length

```python
print(3 in s)
print(len(s))
```

## Set algebra
Very common in interview logic problems.

```python
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)  # union
print(a & b)  # intersection
print(a - b)  # difference
print(a ^ b)  # symmetric difference
```

## Set comprehension

```python
evens = {x for x in range(10) if x % 2 == 0}
```

## Complexity note
Average-case O(1) for add/remove/membership.

## Interview pitfalls

- Assuming order from set iteration
- Using `remove()` when key might not exist
- Trying to store mutable objects in a set
