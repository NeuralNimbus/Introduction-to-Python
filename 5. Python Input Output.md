# Python Input and Output

## Why input/output is important
Most interview programs read input, process it, and print output.  
So you must be comfortable with parsing and formatting.

## Output using `print()`
`print()` sends data to standard output (usually terminal).

```python
print("Hello")
print("A", "B", "C", sep="-")
print("Done", end="!")
```

Useful parameters:

- `sep`: separator between multiple values
- `end`: what to print at the end (default newline)

## Formatting output
Prefer f-strings in modern Python because they are clear and fast.

```python
name = "Ana"
score = 91
print(f"{name} scored {score}")
print(f"Average: {91.2345:.2f}")
```

## Input using `input()`
`input()` always returns a string.

```python
x = input("Enter a number: ")
n = int(x)
```

## Reading multiple integers

```python
nums = list(map(int, input().split()))
print(nums)
```

## Handling extra spaces safely

```python
raw = input().strip()
if raw:
    n = int(raw)
else:
    n = 0
```

## Interview pitfalls

- Forgetting conversion (`int`, `float`)
- Crashing on invalid integer input
- Not stripping trailing/leading spaces
