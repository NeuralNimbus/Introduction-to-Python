# Basic Python Operators

## What are operators?
Operators are symbols/keywords that perform operations on values (operands).  
Example: in `3 + 4`, `+` is the operator and `3`, `4` are operands.

## Arithmetic operators
Used for numeric calculations.

| Operator | Meaning | Example |
|---|---|---|
| `+` | add | `a + b` |
| `-` | subtract | `a - b` |
| `*` | multiply | `a * b` |
| `/` | true division | `a / b` |
| `//` | floor division | `a // b` |
| `%` | remainder | `a % b` |
| `**` | power | `a ** b` |

```python
print(7 / 2)   # 3.5
print(7 // 2)  # 3
print(-7 // 2) # -4 (floor means round down)
```

## Comparison operators
Comparison operators return boolean values (`True`/`False`).

`>`, `<`, `>=`, `<=`, `==`, `!=`

They are frequently used in `if` conditions and loops.

## Logical operators
`and`, `or`, `not` combine or invert conditions.

```python
age = 20
print(age >= 18 and age <= 60)  # True
```

## Assignment operators
Assignment stores or updates values.

- Basic: `=`
- Compound: `+=`, `-=`, `*=`, `/=`, `//=`, `%=`

```python
x = 5
x += 2
print(x)  # 7
```

## Identity operators
`is`, `is not` check if two variables reference the same object.

```python
x = None
if x is None:
    print("No value")
```

## Membership operators
`in`, `not in` check whether a value exists inside a sequence or set.

```python
print('py' in 'python')  # True
print(3 in [1, 2, 3])    # True
```

## Bitwise operators (less frequent but useful)
`&`, `|`, `^`, `~`, `<<`, `>>` work on binary representations.

## Interview pitfalls

- Confusing `/` and `//`
- Using `is` for value comparison
- Not handling negative floor division correctly
